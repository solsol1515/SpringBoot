[자바의 DB 연동 절차]

2. 연결 객체 얻어오기 (jdbc : Connection, mybatis : SqlSession)
3. sql 문장
4. 전송 객체 (Statement / PreparedStatement)
5. 전송
6. 전송 객체 닫기
7. 연결 객체 닫기

1. 연결 객체 얻어오기 (Connection)
2. sql 문장
3. cursor 객체 얻어오기(자바 전송 객체 얻어오기와 유사)
4. 실행
5. cursor 객체 닫기
6. 연결 객체 닫기
